# Оно тебе надо

Учебный проект Яндекс Практикума — лендинг вымышленного аукциона вещей, в которые никто не верил.

## Стек

- HTML5
- CSS3

## Структура проекта

```
ono-tebe-nado/
├── index.html          — главная страница
├── README.md           — документация
├── images/             — изображения и иконки
├── fonts/              — шрифты и fonts.css
└── styles/
    ├── global.css      — глобальные стили и сбросы
    └── style.css       — стили компонентов страницы
```

## Секции страницы

- **Header** — навигация, логотип, контактная информация
- **Cover** — главный баннер с заголовком и кнопкой
- **Lots** — карточки лотов аукциона
- **About** — информация о проекте
- **Footer** — контакты, навигация, ссылки на соцсети

## Как открыть

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере

## Примечание

Проект выполнен в рамках курса Яндекс Практикума по верстке.
